Smart, Compensation-Tracking, Human Energy Harvesting System

Overview:

Exercise device that generates electric power, generating financial compensation for the User. Users connect to the system via a smartphone so that the system can identify them to record the revenue generated to a Personal Energy Account (PEA), and display real-time data on their Smartphones. Energy generated from use of this system can be directed to either the grid or stored in a battery bank for powering a conventional electric or fuel-cell vehicle. In addition, the device has an Auxiliary Power Outlet, which enables Users to power other electric devices.

Typical Exercise Session:

At the start of each session, Users connect their Smartphones via WiFi to the Human Energy Harvester (HEH) by scanning a QR code (on a display on the Energy Monitor Interface) by using the Smartphone’s camera. This connects to an application on a remote server, and starts the session via a browser on that Smartphone.

The first time that Users use this system, they will be prompted to enter their names, and will then automatically be taken to the main live-data screen. If the Users have been on the system previously, they will automatically see the live-data screen. The live data screen displays real-time session information. The session begins when the User starts pedaling (detected by the Energy Monitor Interface), and continues for a pre-determined period of time, say max 30 minutes. Optionally, the User may end the session early by pressing a button on the Smartphone.

The real-time session screen displays the following on the User’s Smartphone:

  • elapsed time
  • instantaneous power being generated
  • average power generated
  • total energy generated for the session
  • local grid’s clean power buy-back rate
  • total financial compensation generated for the session
Electronics within the device measure the voltage and current dynamically and calculate the power generated. When the session ends, a button on the Smartphone display enables Users to view their historical statistics, including:

  • total number of sessions and calendar days since first use
  • average power generated per session
  • total energy produced since first use
  • financial compensation generated since first use
Prior to the start of any session, Users can direct where they want their power to go, including:

  • the local grid
  • battery bank (for conventional electric vehicle charging)
  • electrolyzer (for hydrogen fuel-cell vehicle use via battery bank)
  • auxiliary power outlet (for powering backup devices)
